## Prerequisites

- Nodejs v8
- yarn / npm

## Install

```shell
    npm install
```

## development

```shell
    npm run dev
```

## test

```shell
    npm run test
```

## test with cypress

open terminal 1

```shell
    npm run dev

```

open terminal 2

```shell
    npm run cypress

```

## build

```shell
    npm run build
```

## prod

```shell
    npm run prod
```
